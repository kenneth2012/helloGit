# 文档更新

文档和代码映射更新指南。

## 何时更新文档

1. **功能变更** - 添加、修改或删除功能
2. **API 变更** - 添加、修改或删除 API
3. **配置变更** - 修改配置项
4. **架构变更** - 修改系统架构

## 文档类型

### README
- 项目概述
- 安装说明
- 快速开始
- 使用示例
- 贡献指南

### API 文档
- 端点列表
- 请求/响应格式
- 认证说明
- 错误码

### 代码文档
- 文档字符串
- 注释
- 类型提示

## 更新流程

```
1. 识别变更
   ↓
2. 评估影响
   ↓
3. 更新相关文档
   ↓
4. 检查一致性
   ↓
5. 代码审查
   ↓
6. 提交
```

## 文档质量标准

| 标准 | 要求 |
|------|------|
| 完整性 | 包含所有必要信息 |
| 准确性 | 与代码一致 |
| 清晰度 | 语言清晰易懂 |
| 可维护性 | 易于更新 |
