代码重构和清理指南。

## 重构清单

- [ ] 未使用的变量

### 2. 代码质量
- [ ] 长函数拆分
- [ ] 大类拆分

- [ ] 条件简化

### 3. 导入清理
```python
# 未使用的导入
import os  # 未使用

# 应该删除
```

### 4. 变量清理
```python
# 未使用的变量

# 应该删除或使用

print(result)
```

## 重构流程

```
1. 运行静态分析
   ↓
2. 识别问题
   ↓
3. 小步重构
   ↓
4. 运行测试
   ↓
5. 代码审查
   ↓
6. 提交
```

## 常用重构模式

| 模式 | 描述 | 适用场景 |
|------|------|----------|
| Extract Method | 提取方法 | 函数过长 |
| Extract Variable | 提取变量 | 表达式复杂 |
| Rename | 重命名 | 命名不清 |
| Inline | 内联 | 过度抽象 |
| Move | 移动 | 位置不当 |
| Extract Class | 提取类 | 类过大 |
